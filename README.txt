En este archivo zip se encuentra la sección de código del proyecto.

Esta sección se divide en 3 partes:


Generacion y Preprocesamiento dataset: Esta carpeta  se corresponde con la obtención de los datos 
y generación de los datasets utilizados para el cálculo de predicciones. 

Está contenido en un notebook de Python en la carpeta Obtención de datos y Generación de datos.
Dicho notebook contiene la extracción de los datos de los ficheros csv del dataset público
y la generación de los datasets con los datos obtenidos mediante la técnica del Web Scrapping

Para su correcto funcionamiento en la misma carpeta se incluyen 4 ficheros csv correspondientes al dataset público
llamados season-1516_csv.csv, season-1617_csv.csv, season-1718_csv.csv, season-1819_csv.csv



Descripcion Dataset Final:  Esta carpeta contiene un notebook llamado Descripcion Dataset Final 
que contiene una descripción de las principales características del dataset obtenido.


Predicciones: Esta carpeta contiene los notebooks con las diferentes predicciones realizadas con los diferentes modelos.
